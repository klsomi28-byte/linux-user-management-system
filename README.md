# User Management System (Linux)

## Description
A simple Linux Bash script to manage system users.

## Features
- Add a new user
- Delete an existing user
- List all users

## Operations
- Create user using `useradd`
- Delete user using `userdel`
- View users from `/etc/passwd`

## How to Run
```bash
chmod +x user_management.sh
sudo ./user_management.sh

##Requirements
Linux OS
Root (sudo) privileges
Bash shell
